蜂鸟
===============

> `蜂鸟`是一个图像(jpg/png/webp/svg/gif)压缩客户端。

##Install

* <a href="https://github.com/stormtea123/hummingbird/releases">OS X</a>
* <a href="https://github.com/stormtea123/hummingbird/releases">Windows</a>
